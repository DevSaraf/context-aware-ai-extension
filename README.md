# Context Aware AI Extension

A browser extension that adds persistent context memory to LLM conversations.

## Problem
LLMs forget previous company decisions and context across conversations.

## Solution
This extension creates a memory layer that:
- Detects prompts in real time
- Retrieves relevant context
- Displays suggestions via sidebar

## Features
- Prompt detection inside ChatGPT
- Context-aware sidebar
- Memory-driven suggestions
- Chrome extension architecture

## Future Work
- Semantic search
- Vector memory
- Multi-source ingestion (Slack, Docs, Tickets)

## Vision
Become the memory layer between companies and AI systems.
